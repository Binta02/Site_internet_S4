Ce projet consiste en {\bfseries{un travail d\textquotesingle{}audit et de debug de site web}} pour un client ayant une entreprise de prestation en régie. Notre objectif est d’améliorer le visuel du site web afin de le rendre plus moderne et agréable (frontend) mais aussi son fonctionnement afin de le rendre plus fluide et fonctionnel (backend). Ce site web doit être responsive et offrir certaines fonctionnalités aux clients selon leurs rôles. Ces différents rôles sont visibles dans la base de données.\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Accès au projet}{Accès au projet}}\label{index_autotoc_md4}
