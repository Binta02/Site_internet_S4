\doxysubsection{Description détaillée}
Classe abstraite \doxylink{class_controller}{Controller}. 

Cette classe sert de base pour tous les contrôleurs. Elle gère les actions et le rendu des vues. Les classes filles doivent implémenter la méthode {\ttfamily action\+\_\+default}.
